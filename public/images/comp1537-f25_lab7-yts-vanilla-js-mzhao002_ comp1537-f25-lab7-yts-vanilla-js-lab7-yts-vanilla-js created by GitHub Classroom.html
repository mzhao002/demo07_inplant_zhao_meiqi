      Fetching Movies from the YTS API using Vanilla JavaScript

      Objectives:

      1. Replace jQuery AJAX with the Fetch API for HTTP requests.
      2. Dynamically create and append HTML elements to display data.
      3. Add functionality to load additional content (pagination).
      4. Understand asynchronous operations and error handling in JavaScript.

      Introduction

      Now you have mastered jQuery 😅, let us move on to Vanilla JS. Vanilla JavaScript refers to using plain JavaScript without any additional libraries like jQuery. It it called "vanilla" because it is the most basic form of JavaScript, without any added flavors or enhancements. While jQuery was popular for simplifying tasks in JavaScript, modern JavaScript (ES6+) has introduced features that make jQuery less necessary. Using Vanilla JavaScript directly can improve performance, reduce file sizes, and eliminate dependency on external libraries. This lab guides you in fetching and displaying movies from the YTS API using Vanilla JavaScript, leveraging features such as fetch and document methods.

      https://illustrious-blini-b80629.netlify.app/

      Part 3: Displaying Movies on the Page

      1. Process and Display Movie Data:

        • Define a process_response function to build the HTML for each movie, including the movie title, rating, and cover image:
        function process_response(aResponse) {
  console.log(aResponse);
  result_html = "";
  result_html += `<div class="flex flex-wrap gap-2 justify-center max-w-screen-lg mx-auto">`;
  for (let i = 0; i < aResponse.data.movies.length; i++) {
    result_html += `
      <div class="basis-1/4">
        <div class="relative">
          <img src="${aResponse.data.movies[i].large_cover_image}" alt="" />
          <div class="text-white absolute left-1/2 top-1/2 -translate-x-1/2 -translate-y-1/2">
            <svg xmlns="http://www.w3.org/2000/svg" class="text-green-400 icon icon-tabler icon-tabler-star-filled" width="44" height="44" viewBox="0 0 24 24" stroke-width="1.5" stroke="#7bc62d" fill="none" stroke-linecap="round" stroke-linejoin="round">
              <path stroke="none" d="M0 0h24v24H0z" fill="none"/>
              <path d="M8.243 7.34l-6.38 .925l-.113 .023a1 1 0 0 0 -.44 1.684l4.622 4.499l-1.09 6.355l-.013 .11a1 1 0 0 0 1.464 .944l5.706 -3l5.693 3l.1 .046a1 1 0 0 0 1.352 -1.1l-1.091 -6.355l4.624 -4.5l.078 -.085a1 1 0 0 0 -.633 -1.62l-6.38 -.926l-2.852 -5.78a1 1 0 0 0 -1.794 0l-2.853 5.78z" stroke-width="0" fill="currentColor" />
            </svg>
            ${aResponse.data.movies[i].rating}
          </div>
        </div>
        ${aResponse.data.movies[i].title}
      </div>`;
  }
  result_html += `</div>`;
  result_div = document.getElementById("result");
  let x = document.createElement(`div`);
  x.innerHTML = result_html;
  result_div.appendChild(x);
}

      Part 4: Adding a "View More" Button

      1. Creating and Styling the Button:

        • Create a "View More" button, styled with Tailwind classes, and append it after the movie list:
        function process_response(aResponse) {
  // ... previous code

  // Creating "View More" Button
  let y = document.createElement(`div`);
  let z = document.createElement(`button`);
  z.innerHTML = "View More!";
  z.classList =
    "bg-green-500 text-white rounded py-2 px-4 cursor-pointer text-center";
  y.appendChild(z);
  y.classList = `text-center my-4 `;
  result_div.appendChild(y);
}

      2. Event Listener for Pagination:

        • Add an event listener to fetch more movies when the "View More" button is clicked:
        z.addEventListener("click", () => {
  fetch(
    "https://yts.mx/api/v2/list_movies.json?genre=Animation&minimum_rating=8&page=2"
  )
    .then((resp) => resp.json())
    .then((resp) => {
      process_response(resp);
    })
    .catch((err) => {
      console.log(err);
    });
});

      3. Handling Existing Button Removal:

        • Ensure the "View More" button is removed and re-added correctly upon each new fetch to prevent duplicates:
        let tmp = document.getElementById("viewMoreBtn");
if (tmp) tmp.remove();

      Part 5: Global Variable for Pagination

      1. Define and Update Page Number:

        • Create a global variable, page_number, to keep track of pagination and increment it with each fetch:
        let page_number = 1;

z.addEventListener("click", () => {
  fetch(
    `https://yts.mx/api/v2/list_movies.json?genre=Animation&minimum_rating=8&page=${page_number}`
  )
    .then((resp) => resp.json())
    .then((resp) => {
      process_response(resp);
    })
    .catch((err) => {
      console.log(err);
    });
  page_number++;
});
